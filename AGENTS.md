# Helios 44-2 Adapter Project

This project is an attempt to programmatically design an adapter for the Helios 44-2 lens so that it can be used on a Leica M camera - with rangefinder coupling, so focusing works correctly.

## Research

A initial research document has been provided in the `docs/research.md` file.

## Skills

Various essential skills are required to complete this project, as Claude was not made to work with CAD software -- therefore, make sure to use the skills included with this project.
